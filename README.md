# poe2-trade2

## 项目名称: 流放2网购助手

## 目标:
实现一个流放之路2网页集市的chrome插件,参考网址如下:https://poe.game.qq.com/trade2/search/poe2/%E6%B7%B1%E6%B8%8A%E5%B4%9B%E8%B5%B7
### 实现内容:
整体UI 界面采用 Sidebar UI,使用中文语言, 显示并占据整体页面的最后右侧, 可收缩成一个右侧的小图标,点击打开 sidebar ui , sidebar ui关闭后,只保留这个小图标.
### 具体功能如下:
1. 物品收藏功能 ,物品收藏可以手动创建和删除目录,目录采用树型显示. 在集市物品的最左侧排列 收藏按钮,显示为一个五角星. 点击收藏按钮,默认进行当前选择目录下,收藏内容为物品属性/通货价格等信息. 并能触发点击 Travel to Hideout 按钮的功能.
2. 搜索存储,同收藏功能,可以手动创建和删除目录,目录采用树型显示. 将当前页面地址保存在当前目录中,并可以进行备注和删除,保存的搜索默认显示备注内容.
3. 收藏的物品提供查找相似功能,在非搜索页面,也能快速找到与收藏物品相似词条的物品.


readme:

流放2国服集市助手,PoE 2 Trade Helper 是一款专为《流放之路 2》国服（poe.game.qq.com）打造的综合浏览器扩展。它旨在增强并简化你的交易体验，直接与官方交易平台集成，提供一套强大的工具，让玩家能够高效管理挂单、筛选交易目标，并轻松追踪游戏内的贵重物品。这款扩展超越了简单的叠加功能，嵌入到官方市场界面中，提供无缝、原生的增强体验，让交易更高效、更有条理。

功能:
1. 收藏集市搜索到的物品,保存到右侧物品收藏中.
2. 收藏集市搜索结果,保存搜索筛选条件,以便下次使用.
3. 收藏的物品可以快速进行所在藏身处
4. 收藏的物品提供查找相似功能,在非搜索页面,也能快速找到与收藏物品相似词条的物品.

PoE 2 Trade Helper 的核心功能是在每个交易结果条目里植入便捷的 “星形收藏” 按钮。只需点击星形图标，玩家就能立即将该物品添加到个人购物收藏中。这省去了使用外部笔记或电子表格记录的麻烦，让所有重要的交易目标都集中保存在浏览器环境内。扩展的侧边栏配备了强大的树形结构组织系统，用户可创建文件夹和嵌套分类，完美适配各类交易策略 —— 无论是倒卖通货、搜寻稀有装备，还是追踪圣所效果与技能词缀。
该扩展具备智能重复检测功能：若你尝试添加已存在于收藏中的物品，它会即时发出提醒，避免收藏列表杂乱冗余。它还会自动监控交易页面的 DOM 实时更新，确保星形按钮等辅助工具在新搜索结果加载或页面发生变化时始终正常显示。这种动态交互设计让你无需手动刷新页面，就能保持交易操作的专注度与效率。此外，PoE 2 Trade Helper 支持多语言物品数据，内置精心整理的英文和中文数据库（en.json 与 cn.json），可精准呈现物品描述、词缀信息并保障搜索功能的准确性，适配不同语言习惯的用户群体。
用户可借助侧边栏的拖放功能，按物品类型、赛季、价值或个人偏好灵活整理收藏；快速移除选项能让收藏列表始终保持精简、实用；轻量化的设计则确保扩展运行流畅无延迟。数据存储完全依托本地浏览器，不进行任何不必要的数据传输，充分保障你的隐私。安装流程简单便捷，仅需授予标准的 Chrome 浏览器权限，扩展会在交易页面加载时自动激活，无需复杂的手动设置。
核心功能：
每个交易结果行嵌入集成式 “星形” 按钮，轻松添加心愿单、构建收藏列表。
侧边栏采用树形视图布局，支持文件夹与嵌套分类，可自定义收藏组织方式。
多语言支持，内置精校的英文、中文物品数据，实现精准追踪与搜索。
拖放文件夹管理，可按用户自定义规则灵活、直观地分类物品。
快速移除控件，可取消物品收藏，保持收藏列表整洁、重点明确。
轻量化且运行高效，依托本地浏览器存储数据，安全无延迟。
